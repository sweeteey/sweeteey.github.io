vscode远程连接服务器

I. 下载vscode

参考博客：https://blog.csdn.net/weixin_46474921/article/details/132841711

II. 安装ssh插件

  • 选择左侧Extensions选项卡，输入框搜索remote，选择安装Remote-SSH插件

  • 安装完成后会在左侧新增一个选项卡Remote Explorer

III. 添加服务器连接配置

  • 点击选项卡，添加ssh连接

  • 输入远程服务器的地址，账号和ip根据自己情况进行修改

  • 保存后，会将刚才的连接信息存储在C:\Users\sweeteey\.ssh\config

IV. 连接服务器

  • 右键选择远程服务器

  • 选择服务器类型

  • 输入密码

可能会出现的问题

I. 多设备冲突

VS Code 远程连接服务器时，会在该服务器的用户目录下（例如 ~/.vscode-server）安装服务端组件、存储配置和扩展信息。当多台电脑使用同一个用户名连接同一台服务器时，它们会尝试读写这个共享的目录

解决方法：为每台电脑配置独立的安装路径

  1. 在当前使用电脑的VSCode中，打开设置
  2. 搜索 remote.SSH.serverInstallPath
  3. 点击 “Add Item” 按钮
  4. 在 Item 字段中，输入在SSH配置文件中使用的主机别名（Host）（例如 10.102.64.48）
  5. 在 Value 字段中，输入在服务器上存放这台电脑专属VS Code Server的绝对路径（例如 /home/cyw/.vscode-server-Win）

II. –no–config

解决方法：手动安装VS Code Server

  1. **获取 Commit ID：**在本地 VSCode 中，点击菜单栏的 帮助> 关于，在列表中找到 提交 后面的一长串字母数字组合（例如 0f0d87fa9e96c856c5212fc86db137ac0d783365），这就是 Commit ID，把它记下来
