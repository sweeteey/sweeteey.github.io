服务器上下载PHPGAT并配置环境

I. git clone

II. 创建虚拟环境

前置问题：

  • PHPGATv2.yaml中python版本和实际服务器python版本不一致
  • PHPGATv2.yaml中torch对应的cuda版本和实际服务器上cuda版本不一致

步骤1：创建基础环境（跳过torch安装）

  1. 复制原YML文件并删除Torch相关依赖

  2. 创建环境（自动安装非Torch依赖）

步骤2：降级python版本

  1. 降级python版本

  2. 检查是否降级成功

步骤3：手动安装适配的PyTorch生态

  1. 卸载所有冲突包

  2. 验证无残留CUDA 11 包

  3. 安装正确版本PyTorch

  4. 验证torch是否可用

步骤4：环境功能验证

保存为env_test.py并运行python env_test.py

成功预计输出

配置服务器

  1. 打开pycharm，找到设置(Ctrl+Alt+S) - 项目 - Python解释器 - 添加解释器 - 基于SSH

  2. 新建SSH连接

  3. 选择服务器的远程 Python 解释器路径，可用which python查看
